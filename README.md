## TheCareCircle
Web version of my app that helps families and home health aids communicate to better help a person in need of extra care

# Project details
- Allow a user to sign up/in and create a "Care Circle" to become an Owner of that Circle.
- Allow Circle Owner to invite other users to that "Care Circle" and change their       permissions to Co-Owner or admin, or keep them as a member if preferred.

# Requirements 
- Homepage with a button to create a "Care Circle" as well as an about section.

- Login/register buttons that require a user to sign up using 
    - Email
    - Password
    - and give a nickname (ideally their real name) that will show in the members

- Calender page that allows members to add events to the calender as well as details like:
    - date of event.
    - time of event.
    - location of event.
    - notes of event.
    - also allow other members to claim "responsibility" of an event so other members know that it will be taken care of.

- Messaging page that allows members in a "care circle" to chat about new events, changes to care, etc.

- Care plan page that allows members to add:
    - Medications, as well as dosages, date and time of required administration.
    - Notes on care plan and update notes as needed.

- Checklist page that allows members to create and edit tasks, to do's, also show which member completed the task.

- Page to where members can add Healthcare providers and contact information, as well as hospitals they can be found.

- Owner only page that allows owner to manage active members, and admins.

# Technology

- Javascript
- React.js
- Node.js
- CSS for styling 
- Vite
- Zustand
- also using Postgres.sql database for data storage
